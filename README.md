# pete.sh

```bash
yarn install
```

```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## stack

frontend

- Next.js
- React
- Prisma
- Tailwind

"backend" (serverless)

- [Vercel edge functions](https://vercel.com/features/edge-functions)
- [Planetscale mySQL database](https://planetscale.com/)
